Making images accessible

Text alternatives (alt text)

For people with access needs to be able to use images, they need a text alternative that describes the information or function conveyed.

This helps people using screen readers, as the text alternative can be read aloud. It also has other benefits, like making images become indexable by search engines, or acting as a fallback if an image can't be loaded.

Images of text

Avoid images that contain text, because the text becomes unreadeable when the user zooms in.

It's good practice to use actual text, that is styled with CSS, raher than image-based text.

Genuine text can be resized without losing clarity or getting distorted.

Users can also modify genuine text to be a certain font size, colour or font family, to suit their needs. This is useful, for example, to people with low vision, or with cognitive disabilities that affect reading.
